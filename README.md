# Algorithm Visualizer

An educational project that visually demonstrates how common algorithms work step by step, making it easier to understand core DSA concepts.

## Features

* Step-by-step algorithm visualization
* Beginner-friendly
* Useful for learning algorithms

## Tech Stack

* Java
* Apache NetBeans
* Java Swing

## How to Run

1. Clone the repository
2. Open in Apache NetBeans
3. Build and run
